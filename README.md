# Web Vulnerability Scanner

Educational web vulnerability scanner with modern UI.

## How to Run (Windows)

1. Open PowerShell or Command Prompt
2. Navigate to the folder:
   ```powershell
   cd C:\Users\karth\OneDrive\Documents\web_vuln_scanner_fixed
   ```

3. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```

4. Run the app:
   ```powershell
   python app.py
   ```

5. Open browser → http://127.0.0.1:5000

## Features
- Modern dark UI
- Security headers check
- Form detection
- Simulated XSS check
- JSON report download
- Print to PDF option
